# Docker Monitor Dashboard

Real-time monitoring dashboard for Docker containers with webhook notifications.

## Project Goals
- Monitor Docker container status in real-time
- Display CPU, memory, and network usage
- Send webhook notifications on container failures
- Clean web interface with live graphs

## Tech Stack
- Python (Flask/FastAPI)
- Docker SDK for Python
- Webhook integration (N8N/Slack)
- Real-time updates

## Development Timeline
- Week 1-2: Basic Docker API integration
- Week 3-4: Dashboard UI
- Week 5-6: Webhook notifications + testing
- Week 7-8: Documentation + deployment
